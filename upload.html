Syarat dan Ketentuan Unggah File:

Untuk memastikan data Anda dapat diproses dengan baik, harap perhatikan syarat dan ketentuan berikut:

  • File harus berformat CSV (Comma Separated Values)
  • File harus memiliki kolom wajib: "tanggal", "populasi", "spl", dan "klorofil"
  • Kolom "tanggal" harus menggunakan format YYYY-MM-DD (contoh: 2023-01-01)
  • Kolom "populasi" harus berisi angka yang menunjukkan jumlah ikan
  • Kolom "spl" harus berisi nilai numerik untuk Sea Surface Level (permukaan laut)
  • Kolom "klorofil" harus berisi nilai numerik untuk konsentrasi klorofil
  • Data harus diurutkan berdasarkan tanggal secara kronologis
  • File tidak boleh memiliki baris kosong di antara data
  • Ukuran file maksimal adalah 10MB

Berikut adalah contoh format tabel yang sesuai dengan syarat di atas:

tanggal populasi spl klorofil ...(kolom opsional lainnya)
2023-01-01 150 25.5 12.3 ...
2023-01-02 155 26.2 13.1 ...
